# cex-auth-service 模块业务功能清单

## 模块定位

认证与安全中心，负责用户登录链路、设备安全和会话管理。

## 必做功能

- 注册登录（邮箱/手机/密码）
- 登录二次验证（Google Authenticator、短信、邮箱）
- 会话管理（token签发、刷新、吊销、设备会话）
- 安全策略（失败锁定、异地登录提醒、高风险登录挑战）
- 防钓鱼码与登录通知
- API Key 管理（创建、权限、IP白名单、禁用）
- 子账户登录与权限委托认证
- 密码找回与安全验证流程

## 后续扩展

- Passkey / WebAuthn
- 企业 SSO（SAML/OIDC）
- 设备指纹与行为风控评分
